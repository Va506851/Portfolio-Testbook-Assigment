What web-designers do?

Web design identifies the Goal of a website of webpage and promotes accessibility for all potential user. This process involves organizing content and images across
a serise of pages, intergrating application and other interactve element

The professionals who perform this process are called web designers, and their job includes the following duties:



  • Selecting easy-to-read fonts
  • Choosing attractive color schemes that also enable easy-to-read fonts
  • Implementing a brand's identity into the colors,fonts and layout
  • Creating a map of the website's structure to ensure intuitive navigation
  • Placing images,logos,text as HTML and CSS, to create layouts and to style pages
  • Making optimized versions of website and pages both for Desktop and Mobile viewing
